# Epic 02 manual test

1. Enable audio.
2. Select each voice and use **Play short note**. Square, triangle, and sawtooth should share a pitch; noise should be distinct.
3. Press and hold, then release. The voice should end cleanly without an obvious click.
4. Use **Play in 0.5s**. The voice must not begin immediately.
5. Rapidly trigger notes for 30 seconds. The interface must remain responsive and the active count must return to zero.
6. Press and release the hold control more than once. A repeated release must not throw or create sound.
